# 08 — Lab Exercises

Complete these labs in order. Each builds on the previous.

| Lab | Topic | Estimated Time |
|---|---|---|
| [Lab 01](lab-01-setup.md) | Install Terraform & AWS CLI | 20 min |
| [Lab 02](lab-02-first-resource.md) | Create your first resource (S3 bucket) | 30 min |
| [Lab 03](lab-03-variables.md) | Use variables and outputs | 30 min |
| [Lab 04](lab-04-cloud-deploy.md) | Full cloud deployment (EC2 + VPC) | 45 min |

> ⚠️ Always run `terraform destroy` at the end of each lab to avoid cloud charges.
